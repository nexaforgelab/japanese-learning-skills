# English Skill Catalog

This catalog lists all 100 skills by learning goal. Start with an entry skill such as `jp-learning-coordinator`, `jp-learning-companion`, `jlpt-coordinator`, or `jp-travel-survival`, then route to specialist skills for daily work.
