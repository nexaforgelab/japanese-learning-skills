# 中文 Skill 目录

本目录按学习目标整理 100 个 Skill。建议先选一个“入口型” Skill，例如 `jp-learning-coordinator`、`jp-learning-companion`、`jlpt-coordinator` 或 `jp-travel-survival`，再按当天任务调用专题 Skill。
