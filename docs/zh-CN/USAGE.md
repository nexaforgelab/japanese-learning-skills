# 中文使用指南

这个仓库是一套面向日语学习的 AI Skill 库。它不是一本固定顺序的教材，而是 100 个可组合的学习代理。你可以把它理解成一个日语学习团队：有总教练，有五十音老师，有词汇老师，有语法纠错老师，有口语陪练，也有 JLPT 备考规划师。

## 适合谁

- 零基础学习者：从五十音、平假名、片假名开始，逐步进入词汇和基础句型。
- 忙碌职场人：把通勤、午休、睡前碎片时间组织成稳定计划。
- JLPT 备考者：按 N5 到 N1 分级学习词汇、语法、听力、阅读。
- 赴日旅行者：重点练便利店、餐厅、车站、酒店、电话等生存场景。
- 商务学习者：训练会议、邮件、电话、敬语、面试和客户沟通。
- 中高级学习者：通过新闻阅读、写作纠错、复述训练提升表达深度。

## 安装方式

如果你的工具支持 Trae 风格的 skill 目录，请把 `.trae/skills/` 放在项目根目录。每个 skill 都是一个单独目录，里面有 `SKILL.md`。

仓库同时提供 `skills/` 镜像目录，方便阅读、复制和二次改造。两份目录内容一致。

## 基本调用方法

你可以直接点名 skill：

```text
请使用 jp-restaurant，训练我在拉面店用日语点餐。
我的水平是 N5，只会简单句，请一步一步问我，不要一次给完答案。
```

也可以先让协调型 skill 制定计划：

```text
请使用 jp-learning-coordinator。
我的目标是 6 个月内能进行日本旅行和基础职场沟通。
每天最多 20 分钟，请帮我安排本周学习计划，
并指定每天应该调用哪些 skill。
```

## 推荐的学习闭环

一次完整学习不要只看讲解，建议按下面闭环使用：

1. 用协调 skill 读取目标、水平、时间和薄弱点。
2. 用专题 skill 推送少量新知识。
3. 让用户回答、造句、跟读、选择或复述。
4. 用纠错 skill 给反馈，标出自然表达和错误原因。
5. 把错词、错句或薄弱语法加入复习队列。
6. 用 `jp-learning-report` 每周复盘。

通用 prompt：

```text
请把这次学习做成一个闭环：
先复习我上次错的 3 个点，再教 1 个新知识点，
然后让我完成 2 个互动任务。等我回答后再纠错，
最后给出下次复习安排。
我的水平是 N4，今天只有 15 分钟。
```

## 如何选择 Skill

如果你不知道该用哪个，优先从这几个开始：

| 目标 | 推荐入口 |
|------|----------|
| 不知道怎么规划 | `jp-learning-coordinator` |
| 想长期有人带学 | `jp-learning-companion` |
| 只有碎片时间 | `jp-fragmented-schedule` |
| 准备 JLPT | `jlpt-coordinator` |
| 准备旅行 | `jp-travel-survival` |
| 想练口语 | `jp-speaking-practice` |
| 想纠正常错 | `jp-common-mistake` 或 `jp-grammar-error-tracker` |
| 想写邮件 | `jp-email-writing` |
| 想练商务 | `jp-business-conversation` |

## 数据与复习

`japanese_agent_core.py` 提供一个轻量 SQLite 学习核心，可以记录：

- 用户画像：等级、目标、每日时间、兴趣、薄弱点
- 学习事件：练习内容、结果、分数
- 复习项：词汇、读音、意思、例句、下次复习时间
- SM2 间隔重复更新
- 每日计划和周报

运行测试：

```bash
python3 test_all_skills.py
python3 test_deep_skills.py
python3 test_e2e_skills.py
```

## 建议使用习惯

- 每次只学 1 到 3 个新点，不要贪多。
- 每次都必须输出一点东西，例如造句、朗读、复述、选择题。
- 纠错时要求给出“错误原因”和“更自然版本”。
- 职场、旅行、考试目标不同，不要混用同一套节奏。
- 每周至少调用一次 `jp-learning-report`，否则很容易只学习、不复盘。
